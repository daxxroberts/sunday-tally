# Design System

## Theme & Register
Stark, engineered product interface with a marketing landing that relies on aggressive neutrality (grayscale) giving way to highly saturated, purposeful color reveals upon interaction.

## Palette
- **Body Background:** `#FAFAFA`
- **Surface:** `#FFFFFF` (used to pop interactive or elevated components)
- **Ink/Typography:** `neutral-900`
- **Primary Accent:** Indigo (`#4F6EF7`)
- **Secondary Accent:** Cyan (`#06B6D4`)
- **Semantic Data:** Emerald (Growth/Positive), Rose (Decline/Negative), Amber (Warning/Caution)

## Typography
- **Primary Font:** Inter / System Sans (crisp, modern, readable)
- **Scale:** Aggressive hierarchy (e.g. 110px hero titles) with tight tracking (`tracking-tighter`) to enforce the "engineered" feel.
- **Numbers:** Tabular nums / mono-spaced logic for all dashboard metrics.

## Component Patterns
- **Buttons:** Grayscale default (`bg-neutral-900`) that flips to full color (`#4F6EF7`) strictly on hover.
- **Cards/Containers:** Bordered (`border-neutral-200`) with high-blur backdrops.
- **Motion:** Parallax scrolling and smooth 0.5s–0.7s color fades (e.g. grayscale to full color transitions).
